pygame의 기본적인 구조

  1. pygame 선언
  2. pygame 초기화 –> pygame.init()

    • size –> x크기 y크기
    • screen –> pygame.display.set_mode(size)
    • clock –> pygame.time.Clock()
  4. pygame loop
  5. pygame event 설정
  6. pygame 화면 설정
